## 简介

Footprints 是一款 SJMCL 扩展，自动保存你删除的实例为纪念卡片，在首页展示如同一座碑林，记录你的 Minecraft 旅程。

提供两个版本：
- **英文版**（org.yoshino.footprints）
- **中文版**（org.yoshino.footprints.zh）

## 功能特色

- 📖 删除实例时自动保存为纪念卡片
- 🏛️ 首页展示所有已删除实例的历史
- 📤 支持导出 / 导入故事存档
- 🖼️ 保留实例名称、版本、时间等信息
- 🗑️ 支持手动移除纪念卡片

## 使用方法

1. 从 Releases 下载对应的 .sjmclx 文件
2. 打开 SJMCL → 设置 → 扩展管理 → 导入
3. 删除实例后自动生成纪念卡片，在首页查看

## 兼容性

- 最低启动器版本: 1.1.3

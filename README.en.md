## 简介

Footprints is a SJMCL extension that automatically saves your deleted instances as memorial cards on your home page, creating a gallery of stories from your Minecraft journey.

Two versions available:
- **English** (org.yoshino.footprints)
- **中文版** (org.yoshino.footprints.zh)

## Features

- 📖 Auto-save deleted instances as memorial cards
- 🏛️ Browse all deleted instance history on home page
- 📤 Export / import story archives
- 🖼️ Retains instance name, version, timestamps
- 🗑️ Manually remove memorial cards

## Installation

1. Download the .sjmclx file from Releases
2. Open SJMCL → Settings → Extension Management → Import
3. Delete an instance to auto-generate a memorial card, view on home page

## Compatibility

- Minimum launcher version: 1.1.3
